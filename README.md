# vehicle-detection-tracking-pipeline
Overview

project here we implements an end-to-end computer vision pipeline for vehicle and pedestrian detection and multi-object tracking. The system evaluates YOLO and Faster R-CNN detectors integrated with ByteTrack for tracking.

Features

Vehicle and pedestrian detection using YOLO and Faster R-CNN

Multi-object tracking using ByteTrack

Detection model comparison

Bounding box generation and evaluation

Video-based tracking pipeline

Video Input → Object Detection (YOLO / Faster R-CNN) → ByteTrack → Tracking Output
